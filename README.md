# 🌟 Glowish Cosmetics - Premium Landing Page

Welcome to the repository for the **Glowish Cosmetics** official website. This is a luxurious, single-page e-commerce landing page designed to showcase the premium Glowish Face Serum. It features a modern, high-end UI with glassmorphism effects, smooth scrolling animations, and a rich purple and gold color scheme.

## ✨ Features

*   **Luxurious UI/UX:** Deep purple gradients, gold accents, and frosted glass (glassmorphism) containers.
*   **Fully Responsive:** Looks perfect on mobile phones, tablets, and desktop computers.
*   **Interactive Product Gallery:** Users can click through beautiful lifestyle and product shots with smooth transitions.
*   **Scroll Animations:** Elements fade, slide, and zoom into view as the user scrolls, powered by AOS (Animate on Scroll).
*   **Custom Brand Preloader:** A sleek, breathing logo animation that displays while the site loads.
*   **Quick Purchase Links:** Direct integration buttons for WhatsApp, Amazon, and Flipkart.

## 🛠️ Tech Stack

*   **HTML5** - Semantic markup
*   **Tailwind CSS** (via CDN) - For rapid, utility-first styling and animations
*   **Vanilla JavaScript** - For the image gallery, header scroll effects, and preloader
*   **AOS Library** - For scroll-triggered animations
